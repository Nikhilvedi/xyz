## UI Components

### Input Text Editor
- Multi-line text input
- Light gray background
- Rounded corners
- Scrollable for long content

### Parse Button
- Blue background
- White text
- Full width
- Only visible when text is entered
- Rounded corners

### Day Card
- Light gray background
- Rounded corners
- Day label in bold
- Bullet list of exercises
- Padding around content

### Exercise List Item
- Bullet point (â€¢)
- Exercise text preserved as entered
- Left-aligned
- Clear spacing between items

## Color Scheme

- **Primary**: Blue (#007AFF)
- **Background**: White (#FFFFFF)
- **Secondary Background**: Light Gray (#F2F2F7)
- **Text**: Black (#000000)
- **Secondary Text**: Gray (#8E8E93)
- **Borders**: Light Gray Border (#D1D1D6)

## Typography

- **Navigation Title**: Large, Bold
- **Section Headers**: Title 2, Bold
- **Exercise Text**: Body
- **Hints**: Headline, Secondary color
- **Buttons**: Headline, White

## Interaction States

### Button States
- **Default**: Blue background, white text
- **Pressed**: Slightly darker blue
- **Disabled**: Gray background (when no text entered)

### Text Editor States
- **Empty**: Placeholder visible
- **Active**: Cursor visible, keyboard shown
- **Filled**: Content visible
